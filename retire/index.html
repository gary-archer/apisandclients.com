OAuth Developer Blog

Original Content

Originally this domain hosted a blog with posts related to APIs and clients that use OAuth security. The posts explained integrations, with a particular focus on developer setups and reliability.

I no longer host this blog's content online. Instead, see the following O'Reilly book, which I cowrote with my Curity colleagues. It explains a much more complete OAuth architectural journey:

  • O-Reilly: Cloud Native Data Security with OAuth
  • Cloud Native Code Examples

Accessing Original Content

If you have followed a link to one of the blog's posts, you can still access content if you run the blog locally. The blog uses NEXT.js and you can run it at http://localhost:3000 with the following commands. Once the blog loads you can navigate to your link's relative location.

git clone https://github.com/gary-archer/apisandclients.com
cd apisandclients.com
npm install
npm start
